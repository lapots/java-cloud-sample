# Stack

Current application stack in this branch

- Java 8
- Maven 3
- Spring Boot 1.5.6
- Travis-CI
- Heroku (not supported for branch)
- Spring JPA
- Spring AOP
- Spring Actuator
- Project Lombok
- Java Messaging Service
- ActiveMQ
- Redis
- WebSockets
- RabbitMQ
- Slf4j + Logback
